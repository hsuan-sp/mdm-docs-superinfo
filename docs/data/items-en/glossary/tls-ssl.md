## Definition

TLS (Transport Layer Security) and SSL (Secure Sockets Layer) are cryptographic protocols designed to provide communications security over a computer network. They ensure that data transmitted between a client (such as a web browser or an iPad) and a server is encrypted and cannot be eavesdropped upon or tampered with by unauthorized parties. All secure communication between an MDM server and its managed devices relies on TLS encryption.
