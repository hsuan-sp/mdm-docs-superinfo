## Definition

Introduced in iOS 26.2, AirDrop Codes add an extra layer of privacy for file sharing. Users can generate a one-time code (valid for up to 30 days) to share with someone who is not in their contacts. The recipient must enter this code to verify the transfer. This ensures that files are only sent to the intended recipient and prevents unsolicited AirDrop requests.
