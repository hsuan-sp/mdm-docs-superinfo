## Definition

A Firmware Password is a low-level security lock for Intel-based Macs (replaced by "Recovery Lock" on Apple Silicon Macs). Once set, the password is required to boot the Mac from any disk other than the designated startup disk or to enter Recovery Mode. This prevents unauthorized users from bypassing the main login screen to steal data or wipe the computer.
