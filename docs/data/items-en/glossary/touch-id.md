## Definition

Touch ID is Apple's capacitive fingerprint identity sensor, seamlessly integrated into the Home button or Power button. It allows users to securely unlock their devices, authorize Apple Pay transactions, and sign in to various applications. MDM administrators can configure specific security policies that require Touch ID authentication to open sensitive school applications.
