## Definition

Unified Logging is a comprehensive system-wide logging mechanism used by all Apple platforms. It provides a centralized stream of logs from the operating system kernel and all applications. In the event of a system crash or network failure, IT administrators can use the `log collect` command to export these records. Analyzing these logs is the most accurate way to determine the exact timestamp and cause of a technical failure.
