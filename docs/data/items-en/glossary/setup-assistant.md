## Definition

The Setup Assistant is the collection of guidance screens that appear when an Apple device is turned on for the very first time (starting with the "Hello" screen). It guides the user through initial settings like Language, Wi-Fi, Apple Account, and Location Services. By using **Automated Device Enrollment (ADE)**, IT can choose to "skip" most of these screens, allowing a device to go from the box to a ready-to-use state in seconds.
