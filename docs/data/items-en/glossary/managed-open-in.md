## Definition

Managed Open In is a Data Loss Prevention (DLP) feature that allows MDM to control how documents are shared between applications. For example, it can prevent a sensitive school document opened in a "Managed App" (like school email) from being shared or opened in an "Unmanaged App" (like a personal social media app). This prevents institutional data from accidentally leaking onto personal devices or unauthorized platforms.
