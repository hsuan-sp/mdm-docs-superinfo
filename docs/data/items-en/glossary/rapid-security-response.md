## Definition

Rapid Security Response (RSR) is a specialized update mechanism from Apple designed to deliver important security fixes between larger software updates. Unlike a full OS upgrade, RSR files are small, install very quickly, and often do not require a full reboot (or only a standard restart). This allows IT administrators to deploy urgent patches against "Zero-Day" vulnerabilities without disrupting the classroom or enterprise workflow.
