# Q: 如何啟用 iPad Pro 與 Mac 的 Wi-Fi 6E 160MHz 頻寬？為何我的裝置速度沒有提升？

# Answer

**iPadOS 26.2 與 macOS Tahoe 26.2 解鎖了對應硬體的 Wi-Fi 6E 160MHz 頻寬支援，可大幅提升傳輸吞吐量。若速度未提升，通常是 AP 設定或頻段干擾問題。**

## 啟用條件

1. **支援的裝置**：
   - iPad Pro 11"/13" (M2, M4 晶片)
   - MacBook Pro (M2 Pro/Max, M3, M4 系列)
   - Mac mini (M2 Pro, M4)
2. **系統版本**：必須更新至 **iPadOS 26.2** 或 **macOS Tahoe 26.2** 以上。
3. **無線基地台 (AP)**：
   - 必須支援 **Wi-Fi 6E (802.11ax 6GHz)**。
   - 必須在 6GHz 頻段啟用 **160MHz** 頻寬 (由網管人員設定)。
   - 加密模式必須為 **WPA3** (Wi-Fi 6E 規範強制)。

## 速度沒變快的原因與排解

- **原因 1：連到了 5GHz 而非 6GHz**
  - 許多學校採用的 SSID 雙頻合一 (Band Steering)。若 6GHz 訊號較弱，裝置會自動跳回 5GHz。
  - **解法**：建議在教室 AP 將 6GHz 設為獨立 SSID (如 `School-6E`) 供高效能需求使用。
- **原因 2：WPA3 相容性問題**
  - Wi-Fi 6E 強制走 WPA3，若 RADIUS 伺服器老舊不支援，裝置會拒絕連線或降級。
- **原因 3：網路線瓶頸**
  - AP 本身的 Uplink 網孔若只有 1Gbps，就算 Wi-Fi 跑到 2.4Gbps 也沒用。建議升級至 2.5GbE 或 10GbE 交換器。
